Delta Chat bot that allows to create channels out of RSS/Atom feeds.

## Install

```sh
pip install channelsbot
```

Configure the bot:

```sh
channelsbot init dcaccount:nine.testrun.org
```

Start the bot:

```sh
channelsbot serve
```

Run `channelsbot --help` to see all available options.
